# Todo App

A simple vanilla JavaScript todo application for adding, completing, and deleting tasks.

## Files

- `index.html` - App structure and page layout.
- `styles.css` - Styling and responsive layout.
- `app.js` - Todo app logic, local storage persistence, and event handling.
- `package.json` - Project metadata.

## Usage

Open `index.html` in a browser to start using the todo app.

## Features

- Add new todo items
- Mark items complete
- Delete tasks
- Persist tasks in local storage
